# URI Kuali

A tool that makes it easy to embed dynamic course data from Kuali.

## What's new in v1.6.0

- Gets the latest active course if a course number has duplicates 
- Adds an exclude parameter in the shortcode.


## Example Usage

``[courses subject="COM"]``
